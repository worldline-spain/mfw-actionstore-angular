$mfwActions
service in module mfw.business.actionstore

Description

The $mfwActions service acts as an action creator for all your promises and dispatches processes to $mfwActionStore.

Dependencies

  • flux
  • $mfwActionStore

Methods

  • newAction(actionBody)

    This method takes a promise and handles its lifecycle broadcasting store events.

    It dispatches the $mfwActionCreation action handled by $mfwActionStore.
